## 🎯 Question Format

Each question follows this structure:

> **Q1. Which type of bond holds complementary base pairs together in DNA?**
>
> A) Ionic bonds
>
> B) **Hydrogen bonds** ✅
>
> C) Covalent bonds
>
> D) Peptide bonds
>
> *Answer: B | Hydrogen bonds hold A-T (2 bonds) and G-C (3 bonds) pairs together.*
